# MICROPAD-

<img width="577" height="575" alt="Screenshot 2026-06-29 231348" src="https://github.com/user-attachments/assets/d023aeee-4d93-44ab-a708-e8fbc861077a" />

I  built a updated macro pad with 9 switches an 0.91 inch Display , a rotor encoder through which I can change between the layers and one RGB LED per switch , It will be  use wireless connection and will use Xiao nrf52840 as the main microcontroller.

## Features

- Base on Xiao NRF Module
- Has per switch LED
- Works wirelessly and has a Battery built in
- Has 9 Switch and 3 Different Layers to switch onto
